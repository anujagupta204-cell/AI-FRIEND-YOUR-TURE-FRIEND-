## STEP 1: CREATE GITHUB ACCOUNT
Go to https://github.com/signup
- Sign up with email
- Verify email
- Create account

## STEP 2: CREATE REPOSITORY
Go to https://github.com/new
- Repository name: voice-ai-friend
- Description: Interactive AI chat with voice and text support
- Make it PUBLIC
- Click "Create Repository"

## STEP 3: UPLOAD FILES
Option A - Using GitHub Web (Easiest)
1. Go to your new repository
2. Click "Upload files"
3. Drag and drop these files:
   - index.html
   - style.css (or style-prod.css)
   - app.js  
   - README.md
   - package.json

Option B - Using Git Command Line
`
git init
git add .
git commit -m "Initial AI Friend release"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/voice-ai-friend.git
git push -u origin main
`

## STEP 4: ENABLE GITHUB PAGES
1. Go to Repository Settings
2. Scroll to "Pages" section
3. Under "Source", select "main" branch
4. Click Save
5. Wait 2 minutes
